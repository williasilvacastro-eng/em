# VORTEX PRO - BASIC INSTALLATION TUTORIAL

## How to install

1. Download the emulator package.
2. Extract the files to a folder on your PC.
3. Run the emulator as Administrator.
4. Wait while the app automatically checks and installs the required components.
5. Enter your license key on the login screen.
6. Open the emulator and apply your preferred settings.

## Important notes

- Keep the emulator folder in a safe location and avoid moving files after setup.
- On the first launch, Windows may ask for permission to install required components. Accept it to continue.
- If the emulator does not open correctly on the first try, restart the PC and open it again as Administrator.

## Need help?

If you have any installation issue or the emulator does not start correctly, open a support ticket in the Discord server and the staff team will help you.
